The George Moses Horton Project

Examining responsible AI through an interactive, archival case study

How might we use AI to resurrect suppressed histories? The George Moses Horton Project reimagines the 19th-century poet’s voice through an AI-driven, interactive avatar. Visitors can ask Horton questions and receive spoken responses synthesized from historical text and machine learning models, turning archival study into dialogue.

Domain AI · AR · Exhibition Design · Historical Storytelling

Blending responsible machine learning with immersive interpretation inside museum and classroom settings.

Experience Interactive avatar installation

Conversation-led encounters where visitors pose questions directly to a resurrected Horton.

Stack Speech-to-Text · GPT · AWS Polly · D-ID API

Combining language models with voice and facial synthesis for responsive dialogue.

Setting Case study in responsible AI

Designed to interrogate ethics, authorship, and empathy within cultural heritage technologies.

Core components

  • Speech-to-Text + GPT integration: Transforms visitor prompts into historically grounded, context-aware replies.
  • AWS Polly + D-ID API: Generates lifelike vocal and facial expressions for real-time engagement.
  • Gradio front-end + Python stack: Enables accessible interaction for museum and classroom use.

Impact

This prototype reframes AI as a tool for cultural preservation rather than replacement—bridging human memory, algorithmic interpretation, and empathy.
